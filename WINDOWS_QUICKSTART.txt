üöÄ FIRST TIME SETUP (Do this once)

1. Open Command Prompt and navigate to the project folder:
   
   cd path\to\Aquamitra


2. Run the automated setup script:
   
   setup-windows.bat


3. Edit the .env file with your API keys:
   
   notepad .env
   
   Add your keys:
   GROQ_API_KEY=your_groq_key_here
   GOOGLE_API_KEY=your_google_key_here
   HF_TOKEN=your_huggingface_token_here
   
   Save and close.


4. Install frontend dependencies:
   
   cd chatbot-ui
   npm install
   cd ..

‚ö†Ô∏è TROUBLESHOOTING
================================================================================

Problem: "Python not found"
Solution: Install Python 3.11 from https://www.python.org/downloads/
          Make sure to check "Add Python to PATH" during installation


Problem: "Node not found"
Solution: Install Node.js from https://nodejs.org/


Problem: Port 8000 already in use
Solution: netstat -ano | findstr :8000
          taskkill /PID <PID_NUMBER> /F


Problem: Port 5173 already in use
Solution: netstat -ano | findstr :5173
          taskkill /PID <PID_NUMBER> /F


Problem: "Module not found"
Solution: Make sure virtual environment is activated:
          .venv\Scripts\activate
          pip install -r requirements.txt


Problem: Groq API rate limit
Solution: Wait 1-2 minutes or upgrade to Groq Dev Tier
